## Game Description
Tank Brawl is a third-person/top-down 3D tank combat game. Players control a tank and battle against enemies across destructible and open arena environments. The current target is a single-level playable demo featuring tank movement, aiming, shooting mechanics, and basic enemy or obstacle interaction.
